# Code of Conduct

This project follows a simple contributor standard: be respectful, precise, and constructive.

## Expected Behavior

- Use welcoming and inclusive language.
- Focus criticism on code, design, documentation, and reproducible behavior.
- Assume good intent while asking for enough evidence to debug issues.
- Respect different experience levels and time zones.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated disruption of technical discussion.

## Reporting

Report concerns through the repository issue tracker or to the maintainers privately when public discussion would be inappropriate. Maintainers may remove comments, close issues, or block contributors when needed to protect the project.
