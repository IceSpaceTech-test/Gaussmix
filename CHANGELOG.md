# Changelog

All notable changes are summarized from `gaussian_mixture_log.md`.

## v1.0.0 - 2025-11-03

### Added

- Completed production-facing documentation, API reference, release notes, and contribution guidance.
- Added final notes for scikit-learn integration, visualization, benchmarks, and ecosystem compatibility.
- Documented known limitations for full covariance scaling and correlated-data convergence.
- Added roadmap items for neural mixture models, distributed EM, and GPU acceleration.

## 2025 Development

- Added unsupervised anomaly detection with contamination-aware thresholding.
- Added BIC-based feature selection with feature importance scores.
- Added scikit-learn ecosystem integration for pipelines and meta-estimators.
- Added visualization utilities and a benchmark suite covering UCI-style datasets.

## 2024 Development

- Added sparse high-dimensional GMMs.
- Added Kalman/time-series mixture utilities.
- Added hybrid GMM/K-Means clustering.
- Added mixture embeddings and online/streaming EM updates.

## 2023 Development

- Added Bayesian GMMs with Dirichlet weight priors and covariance priors.
- Added Wasserstein distance, entropy, sampling, density quantiles, and robust Huber-loss fitting.

## 2022 Development

- Initialized the project structure and core GaussianMixture API.
- Implemented EM, covariance constraints, initialization, prediction, scoring, model selection, serialization, validation, sample weighting, and tests.
- Added log-space numerical stability utilities and AICc/model-comparison helpers.

## v0.1.0 - 2022-08-26

- First public release with API documentation, usage guide, demo notebook, benchmark script, changelog, MIT license, and code of conduct.
